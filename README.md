This is a simple shopping cart project built with Vanilla JavaScript.

Products are fetched from the FakeStore API and rendered on the page. Users can browse products, search between them, add items to the cart, increase or decrease quantities, and remove products from the cart.

The cart data is stored in LocalStorage, so items remain available after page refresh.

# Features

Fetch products from FakeStore API

Product search

Category filtering

Add / remove items from cart

Increase & decrease item quantity

Total price calculation

Cart data persistence using LocalStorage

User alerts and confirmations using SweetAlert

# Technologies Used

HTML

CSS

JavaScript (Vanilla JS)

FakeStore API

SweetAlert

 ## Notes

The main focus of this project was JavaScript logic and data flow, not advanced UI design.

CSS is kept simple and functional.

This project was built to practice working with APIs, state management, and DOM manipulation.
